# 🧮 Simple Calculator

A basic calculator built using **HTML, CSS, and JavaScript**.
It performs simple arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear (AC button)
* ⌫ Delete last input (DEL button)
* ⚡ Instant result calculation

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 💡 How to Use

1. Open the project
2. Click on numbers and operators
3. Press `=` to get result
4. Use `AC` to clear
5. Use `DEL` to remove last digit
